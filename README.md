⚙️ Como usar

Para rodar o projeto em sua máquina você deve seguir os passos abaixo:

git clone https://github.com/Lucaszanin/shipay-challenge.git

cd nome-do-repositorio
npm install
npm run dev

📖 Sobre

Este projeto foi desenvolvido como parte de um teste técnico da empresa Shipay.

O desafio consistia na criação de um input de busca de produtos, permitindo ao usuário pesquisar dinamicamente itens a partir de uma lista ou API.

A aplicação possui as seguintes funcionalidades:

Buscar produtos por nome
Filtrar resultados em tempo real
Exibir lista de produtos encontrados
Interface simples e intuitiva
📌 Status

✅ Teste técnico concluído

🎯 Objetivo

O objetivo deste projeto foi demonstrar habilidades práticas em desenvolvimento front-end, incluindo:

Manipulação de estado
Criação de componentes reutilizáveis
Consumo e tratamento de dados
Performance em buscas (filtro em tempo real)
🧠 Conceitos utilizados
Componentização
Hooks (useState, useEffect)
Filtragem de dados
Renderização condicional
Boas práticas de UI/UX
💡 Curiosidade

Durante o desenvolvimento, foi possível explorar estratégias para melhorar a experiência do usuário, como:

Debounce na busca (evitando múltiplas requisições)
Feedback visual durante a digitação
Otimização da renderização da lista
